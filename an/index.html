My Works

Web Projet

I developed a planet trajectory generator and displayed it on the web using HTML, CSS, and JavaScript for the front end. This project, my first real end-to-end IT project, allowed me to consolidate my web development skills. To calculate the trajectories of the planets, I implemented two calculation methods, which enabled me to compare the approaches and improve my understanding of scientific computing algorithms. 

Project : 2 people 
Duration: 1 week

AI WEB Big Data Project. 

We carried out a project on the inventory of trees in Saint-Quentin. The first step involved a Big Data phase, where we filtered and visualized the data to retain only the essential information for the subsequent phases of the project. Next, we worked on the AI component, calculating the probabilities of the necessity for felling each tree, as well as identifying the neighborhoods where it would be relevant to replant. Finally, we concluded with the web part, developing a website to present our calculations, methodology, and results clearly and accessibly.

Project : 3 people
 Duration: 3 weeks
